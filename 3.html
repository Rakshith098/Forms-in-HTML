Understanding JavaScript Functions

Functions are one of the core building blocks in JavaScript. A function is a set of statements that performs a task or calculates a value.

Example:

function greet(name) {
            
            return "Hello, " + name + "!";
            
            }
            
            console.log(greet("Alice")); // Output: Hello, Alice!

In the example above, greet is a function that takes a parameter name and returns a greeting string.

Quick Tip

You can also define functions using arrow syntax:

const greet = (name) => "Hello, " + name + "!";
